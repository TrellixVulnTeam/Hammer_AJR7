Welcome to Hammer's documentation!
==================================

Hammer 是一套 **标准化的深度学习模板** ，通过集成常用功能和规范流程，让研发专注于模型/实验设计，并最大限度地保证代码兼容性与不同项目间功能的复用性，提高效率。
